# 钢材表面缺陷检测系统
基于 YOLOv8 的钢材缺陷智能检测可视化工具

## 功能
- 上传图片自动检测6类钢材缺陷
- 实时可视化展示检测结果
- 支持调节置信度、IOU参数
- 基于Streamlit网页界面，一键部署

## 缺陷类别
crazing, inclusion, patches, pitted_surface, rolled-in_scale, scratches
